## 📚 Learnings & Project features

### 🛍️ Product Listing and Details:

Users can see a list of products and view detailed information about each product.

### 🛒 Shopping Cart and Checkout:

Users can add products to their cart and proceed to checkout.

### ❤️ User Wishlist:

Users can add their favorite products to a wishlist for easy access later

### 💸 Order Placement and Payment:

Users can place orders and experience a real-world mimicked payment system.

### 🔍 Order Tracking:

Users can view their placed orders and get updates about their order status.

### 📊 Profile Dashboard:

Users can access their profile dashboard.

### 🔐 User Authentication:

Users can log in and register through traditional email and password, as well as via their Google account by Oauth2.

### 🗣️ Customer Support:

Users can raise a complaint on the contact page.

### 👨‍💼 Admin Capabilities:

Admins can add new products, update existing products, and remove existing products.  
Admins can access their admin dashboard, where they can see logged-in users, current online users, total orders, update orders, and view all products.
